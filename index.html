Matter

This one is my current side project, in this game I'm putting everything I've learned and continue learning about software development, software engineering, game design, pixel-art, animation, project management and even marketing. This game is being developed in Unity, it uses sprites, a basic lighting system made by custom shaders, and handcrafted sprite animations.

Quod's Scape

A prototype I developed to learn about Unity for mobile platforms, and usage of mobile hardware like accelerometer and gyroscope. The main character rotates according to celphone rotation. The map is procedurally generated.

Pixel Hanoi Towers

One of the first projects I made to learn about basic and fundamenteal game development concepts. It was made using HTML5 canvas, without any framework.

Avoid and follow

I'm helping some friends with the gameplay design of this prototype, this one is being developed in Unreal Engine 4 and its meant to be for mobile platforms. The mechanics are as same as Snake, but 3d, having some blocks you can make them to follow you, and others that must be avoided.

3d buildings

Another facet of me is that I love art and graphic design. I participated in a game jam at 2018, my role was as a 3d artist, I had to model, texture and place in a level some buildings to make a city. Here I used Blender, GIMP and Unreal Engine 4 for Physically Based Materials

Port machinery

I participated in a game for a game at 2018 jam, modelling, texturing and placing port machinery, specifically a portainer and a gantry crane. I used Blender for 3d modelling and UV wrapping, Photoshop and GIMP for texturing, and Unreal Engine 4 for designing Physically Based Materials.

Fight arena in cave

At 2016 I participated in a game jam as a 3d artist, I modelled and textured the main stage for the game we were developing, and also modelled some props.
